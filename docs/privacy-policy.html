SyncGrid — Privacy Policy

Last updated: February 15, 2026

Overview

SyncGrid is a Chrome extension that replaces your new tab page with a Speed Dial bookmark manager. We are committed to protecting your privacy. This policy explains what data SyncGrid accesses, how it is used, and what is never collected.

Data We Access

  • Chrome Bookmarks API — SyncGrid reads and writes bookmarks within a dedicated __SyncGrid__ folder in your browser's bookmark tree. This is the sole data source for the extension.
  • Chrome Storage API — User preferences (theme, language, active tab) are stored locally via chrome.storage.local. This data never leaves your device.
  • Favicon API — SyncGrid uses Chrome's built-in favicon service to display website icons. No external favicon service is contacted.

Optional Features

  • AI Title Generation — If you choose to configure an AI provider (OpenAI or Google Gemini), SyncGrid sends only the bookmark URL to the provider's API to generate a title. Your API key is stored locally and is only transmitted to the provider you selected. No data is sent unless you explicitly configure an API key and trigger the feature.
  • Local Folder Sync — If you select a local folder for backup, SyncGrid writes a JSON file to that folder using the File System Access API. The folder path and data stay entirely on your device. If that folder is synced to a cloud drive (Google Drive, OneDrive, etc.), that synchronization is handled by your operating system — not by SyncGrid.
  • Export / Import — You can export your bookmarks as a JSON file and import from one. These files are created and read locally on your device.

Data We Never Collect

  • No analytics, telemetry, or tracking of any kind
  • No browsing history or page content
  • No personal information (name, email, location)
  • No cookies or fingerprinting
  • No data is sent to our servers — we do not operate any servers
  • No third-party scripts, SDKs, or advertising frameworks

Permissions Explained

  • bookmarks — Read and organize your bookmarks within the SyncGrid folder
  • storage — Save your preferences (theme, language) locally
  • favicon — Display website favicons for your bookmarks
  • host_permissions — Limited to api.openai.com and generativelanguage.googleapis.com only. Used exclusively for the optional AI title generation feature when you provide your own API key.

Data Storage

All data is stored locally in your browser:

  • Bookmarks: Chrome's built-in bookmark storage (synced by Chrome Sync if enabled)
  • Settings: chrome.storage.local (device-only)
  • AI API keys: chrome.storage.local (device-only, never synced)

Third-Party Services

SyncGrid contacts external services only when you explicitly configure and use the AI title generation feature:

  • OpenAI API (api.openai.com) — if you select OpenAI as your AI provider
  • Google Gemini API (generativelanguage.googleapis.com) — if you select Gemini as your AI provider

No other external communication occurs.

Children's Privacy

SyncGrid does not knowingly collect any data from children or any other users.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated date. As SyncGrid collects no user data, changes are unlikely to affect your privacy.
